_posts 폴더에 .md 파일로 본문을 작성하고 git으로 아래처럼 push해주면된다.

```ruby
cd C:\github\chu-uj.github.io
git add .
git commit -m "posting"
git push
```



* git에 레파지토리 로컬 저장소로 내려받는법

  ```
  C:\github>git clone https://github.com/chu-uj/chu-uj.github.io.git
  ```
